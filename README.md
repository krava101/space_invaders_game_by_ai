# Space Invaders — браузерна аркада (Single-File HTML)

## 1) Опис гри
**Space Invaders** — класична аркадна гра, де гравець керує космічним кораблем і захищає Землю від хвиль прибульців.

**Геймплей:**
- Гравець рухається по горизонталі внизу екрана та стріляє вгору.
- Вороги (5 рядів × 10 колон) рухаються формацією вліво/вправо та опускаються вниз, коли доходять до краю.
- Прибульці інколи стріляють у гравця.
- Гра завершується, якщо:
  - вороги досягли нижньої межі, або
  - гравець втратив усі життя.
- Вороги **прискорюються**, коли їх залишається менше.

**Система очок та прогрес:**
- Очки нараховуються за знищення ворогів.
- **High Score** зберігається у `localStorage`.

**Апгрейди (випадають з ворогів):**
- **+1 серце** (додає життя, з обмеженням максимальної кількості)
- **Щит на 1 удар** (поглинає наступне попадання)
- **Дробовик** (3 постріли віялом на певний час)
- **Подвійний постріл** (2 паралельні постріли на певний час)

**Керування:**
- `A / D` або `← / →` — рух
- `Space` — постріл
- `Enter` — перезапуск після Game Over
- `M` або клік по індикатору звуку — **Sound ON/OFF**

---

## 2) Як запустити
Проєкт не потребує встановлення.

1. Відкрийте файл **`space-invaders.html`** у будь-якому сучасному браузері (Chrome / Firefox / Safari / Edge).
2. Натисніть будь-яку клавішу або клікніть у вікні гри, щоб почати.

> Примітка про звук: браузери блокують автозапуск аудіо. Тому звук активується лише після першої взаємодії (клавіша/клік). Далі звук можна вимкнути/увімкнути клавішею `M`.

## 4) Процес створення
Робота виконувалась ітеративно відповідно до вимог з `AGENTS.md` (single-file HTML, Canvas, Vanilla JS, Web Audio, localStorage, 60fps).

**Кроки:**
1. **Каркас проєкту**
   - Один HTML-файл з `<canvas>`, HUD-оверлеєм (score/lives/high score) та базовими стилями.
2. **Базова механіка гри**
   - Рух корабля гравця, постріли, обмеження частоти стрільби (cooldown).
   - Генерація ворогів у сітці 5×10.
   - Рух ворогів формацією (лівий/правий напрям) та “drop” вниз на краях.
3. **Ігрові правила**
   - Колізії (AABB): куля-гравець ↔ ворог, куля-ворог ↔ гравець.
   - Життя гравця, умови Game Over.
   - Прискорення ворогів при зменшенні їх кількості.
4. **Візуальні ефекти**
   - Анімований зоряний фон.
   - Частинки вибуху при знищенні ворогів/попаданні по гравцю.
5. **Звук (Web Audio API)**
   - SFX: постріл, вибух, попадання, підбір апгрейдів.
   - Проста фонова “ретро” музика (синтез осциляторами).
   - Дотримання autoplay-політик: ініціалізація аудіо тільки після взаємодії.
6. **Апгрейди**
   - Дроп предметів із шансом при вбивстві ворога.
   - Падіння power-up вниз, підбір гравцем.
   - Реалізація ефектів: додаткове життя, щит на 1 удар, double/spread постріл із таймером.
7. **Перемикач звуку**
   - Тумблер Sound ON/OFF (клавіша `M` або клік по елементу HUD).
   - При вимкненні: зупинка музики та блокування SFX; при увімкненні: відновлення роботи звуку.

**Підхід роботи з AI agent:**
- Генерація первинної версії → перевірка у браузері → уточнення вимог/багів → точкові правки → повторне тестування.
- Ключові зміни (power-ups, shield, toggle sound) додавались окремими патчами, щоб зберігати читабельність і контроль над логікою.
